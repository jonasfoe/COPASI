JC-RM - Introduction and motivation


Developing a model with its initial set of values is just the first step in the modeling process.
Big models that encode full organisms are usually parametrized in many different ways to account for mutations of the organism.

We developed an extension of our modeling editor called Run Manager (JC-RM).
This extension allows the modeler to define a hierarchy of parameter values, simulation and plot settings and couple it with the initial model.
