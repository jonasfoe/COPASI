Version 4.0.19 (development)

Release Date: October 6, 2006

New Features

  • Assignment and ODE rules are now available for global quantities.
  • New optimization methods Nelder & Mead and Simulated Annealing.
  • The user is now able to control the weights used in parameter estimation.
  • The user has now direct control of the starting values for parameter estimation and optimization.
  • Export in XPPAUT format.
  • Improved editing of mathematical expression avoids input errors.
  • The optimization and parameter estimation tasks display now a count of adjustable parameters.
  • Added a message box displaying problems which COPASI may have encountered when starting the GUI application.

Bug Fixes

  • Fixed interface of elementary flux modes to allow scheduling of the task and printing of results including a total count of flux modes.
  • Improved table display of functions, which had line breaks in their mathematical expression.
  • Improved layout of experimental data dialog.
  • Fixed crash caused by committing an invalid expression for a function.
  • Fixed problem assigning kinetic function to reaction with non integer stoichiometry.
  • The display status (shown/hidden) of a curve in a plot is no preserved when a task is rerun.
  • Fixed crash when a product appeared as a parameter for the kinetic function in a irreversible reaction.
  • Fixed crash caused by sliders referring to no longer existing objects.
  • Fixed Madonna export of time parameter for explicitly time dependent models.
  • Improved check before deleting objects like compartments or metabolites to avoid crashes.
  • Fixed crashes related to viewing outdated results, which have been obtained through task execution before the model has been changed.
  • Improved SBML import of function definitions.
  • Fixed international character support in all filenames.
  • Fixed crash of stochastic simulation for empty models.
