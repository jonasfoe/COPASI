COPASI (4.0 Build 15 test version)

Release Date: September 15, 2005

NEW FAETURES

  • There are no new features in this release as this is a bug fix release for Build 14. For new features please see the COPASI (4.0 Build 14 test version).

BUG FIXES

  • Fixed crash in the metabolite table, which occurred when the initial concentration for a new metabolite was changed.
  • Solved problem displaying incorrect function parameters in the function widget.
  • Fixed packaging problem for Windows.
  • Copasi now prompts whether to save a changed file when closing.
  • Optimization returned incorrect results in some particular situations.
  • The progress bar can now be minimized.
  • Fixed crash when clicking on main windows while the progress bar is showing.
