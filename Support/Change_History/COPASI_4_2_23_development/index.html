Version 4.2.23 (development)

Release Date: September 28, 2007

New Features

  • Metabolites may now be changed through arbitrary ODEs and assignments.
  • Compartment volumes may now be dynamically changed through ODEs and assignments.
  • SBML support for import and export of all assignment and rate rules.
  • New tool to check rate laws for inconsistencies and errors

Bug Fixes

  • Fixed bug which changed the performance statistic of the optimization task each time the result was redisplayed.
  • The validity of the objective function for optimization is now checked before task execution to avoid unnecessary task execution.
  • Fixed SBML import problem when encountering negative numbers in assignment or rate rules for model quantities.
  • Fixed SBML export problem which occasionally created rules for unspecified parameters.
  • Fixed several minor problems in the table displays of compartments, metabolites, reactions, etc.
  • Fixed crash which occasionally was triggered by deleting a parameter.
  • Improved results of the Levenberg Marquard optimization method for problems with bounded parameter space.
  • Fixed crash which rarely occurred when executing the convert to irreversible tool.
  • Improved convert to irreversible to tool to better detect the negative and positive parts of a kinetic law.
  • Improved layout of tab widget an Mac OS X
  • Fixed problem on Mac OS X, which prevented to zoom out of plots.
