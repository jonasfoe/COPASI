These error messages can occur when a steady state is calculated.

Identifier
CSteadyState (1)

Message
The model is explicitly time dependent. Therefore, the calculation of a steady state is not very meaning full.

Description
One or more of the kinetic laws used in the model are explicitly time dependent a steady state can therefore not be calculated by COPASI.
