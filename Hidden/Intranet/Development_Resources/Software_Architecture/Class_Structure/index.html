Access of information in COPASI classes

This section describes the data structures that COPASI uses to store information internally. This includes the description of the biochemical model, the tasks, the output definition, etc. This section should be useful for a programmer who wants to access information or create objects in COPASI.
The actual functionality of those classes is described in other sections below.
