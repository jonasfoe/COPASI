Object Classification

Class Objects
Initial Time The initial time of non-autonomous models.
Parameters Initial values of any model entity, which is not of type ASSIGNMENT or TIME, and does not have an initial assignment, or a local reaction parameter.
Observed Constants Initial values of a model entity, which is of type ASSIGNMENT or which are calculated by an initial assignment, and the conversion factor. Matrix elements of the stoichiometry, reduced stoichiometry, and link matrix should be allowed too. But as long as the indexing is not done by name we will not support them.
Variables Transient values of any model entity, which is of type REACTION, ODE, FIXED.
Time The time of the model.
Observed Values Transient values for entities of type ASSIGNMENT, Rates (except for entities of type ASSIGNMENT), reaction fluxes.
Results Numeric values (including vector and matrix elements) calculated by a task.
Numeric Values These are any objects which have an integer or floating point value.
Any Objects All objects.

Implementation

Please note that the rates for entities of type ASSIGNMENT are valid numerical values. They are currently not suppressed by the filter instead they are not listed among the candidates when the tree is populated. In the future we maybe able to calculates these rates the prerequisite would be that we have symbolic differentiation.
