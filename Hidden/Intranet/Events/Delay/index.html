An event delay is the time interval between the firing of an event and the execution of the event assignments. The default delay in COPASI is 0.
The delay can be inserted in two places:
  1. between firing and calculation
  2. between calculation and execution
In COPASI the delay will a have an attribute delayAfterCalculation which determines the insertion point. This attributes translates to the SBML L2V4 attribute of the event element.
